CROSS-FILE CONSISTENCY RULES
When the same organization appears in multiple files, ALL of the following must be identical across every occurrence:
1. Organization name (exact spelling and capitalization)
2. Street address(es) — if an org has moved or has multiple locations, list all locations in every file where the org appears, or clearly label which location is referenced (e.g., "Address (Main Office):" vs "Address (Satellite):")
3. Phone number(s) — use the same primary client-facing number everywhere. If two different numbers appear for the same org in different files, investigate and resolve the discrepancy before publishing.
4. Website URL — use the same canonical URL everywhere. Watch for:
   - .com vs .org vs .net variations (verify which actually loads)
   - Old domain names that redirect or are parked (always use the current working domain)
   - Trailing characters in URLs (especially trailing hyphens on pa.gov URLs, which break links)

URL VALIDATION CHECKLIST
Before adding or updating any URL:
1. Verify the URL actually loads (not a 404, not a parked domain page)
2. Confirm the domain matches the organization (e.g., don't link to lancasterhistory.org for Lancaster Public Library)
3. For pa.gov URLs: check for accidental trailing hyphens (common copy-paste error). Example: "area-agencies-on-aging-" should be "area-agencies-on-aging/"
4. If an org's website has changed domains, update ALL files that reference it
5. Always prefer the organization's own website over "Contact through PA 211" or "Contact through findhelp.org" — only use those as a fallback if no org website exists
6. Prefer direct URLs (e.g., https://ywcayork.org/victim-services/) over generic parent org URLs

COMMON ERRORS TO WATCH FOR
1. Domain confusion: Organizations that have rebranded (e.g., Carlisle C.A.R.E.S. → Community CARES) may have old domains still floating around
2. Stale placeholder URLs: "Contact through PA 211" or "Contact through findhelp.org" when the org actually has a website — always search for the org's actual site first
3. Phone number transposition: Crisis lines are especially important to get right — always cross-reference with the county government or org website
4. Copy-paste URL artifacts: Trailing hyphens, missing trailing slashes, or truncated paths
5. Address drift: Organizations move offices; when updating one file, search for the org name across ALL files and update consistently
